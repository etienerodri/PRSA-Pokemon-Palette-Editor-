# PRSA-Pokemon-Palette-Editor-

What you need to install
To run this tool, you just need Python installed on your computer.

The UI runs on PyQt6, so you'll need to install that library too. Just open your terminal or Command Prompt and run
pip install PyQt6

How to run it
Once Python and PyQt6 are good to go, open your terminal, navigate into the PRSA POKE PALETTE EDITOR folder, and run the main script
python main.py

The editor will pop right up and you're ready to go!
